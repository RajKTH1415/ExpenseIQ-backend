# 💰 ExpenseIQ-backend


## 🚀 Overview

ExpenseIQ-backend is a production-ready RESTful backend application built using Java and Spring Boot for managing personal finances.

The application enables users to securely record income and expenses, manage multiple financial accounts, create budgets, monitor spending patterns, generate reports, and receive notifications about important financial events.

The backend is designed using enterprise-grade architecture, emphasizing scalability, security, maintainability, and clean code practices. It follows layered architecture with modular design, JWT-based authentication, database versioning, centralized exception handling, audit logging, and containerized deployment.

This project simulates the backend of a real-world financial management application similar to Wallet, Money Manager, Walnut, or Google Wallet's expense history.
